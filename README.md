# Cafe Charm Showcase ☕

A modern and responsive cafe showcase website built using React, TypeScript, Tailwind CSS, and Vite.

## 🚀 Live Demo
https://cafe-charm-showcase-csyjb14dr-nikhitha-2006s-projects.vercel.app

## 📌 Features
- Responsive design
- Modern UI/UX
- Interactive sections
- Fast performance
- Mobile-friendly layout

## 🛠️ Technologies Used
- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## 💻 Run Locally

Clone the project:

```sh
git clone YOUR_GITHUB_REPO_LINK
```

Go to the project folder:

```sh
cd your-project-name
```

Install dependencies:

```sh
npm install
```

Start development server:

```sh
npm run dev
```
